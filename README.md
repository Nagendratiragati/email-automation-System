# email-automation-System
Smart Email Automation System using Python (Flask, SMTP) to send, schedule, and manage emails with basic web interface.
## 🚀 Overview

The Smart Email Automation System is a web-based application developed using Python and Flask that allows users to send and schedule emails efficiently. The system uses SMTP protocol for email delivery and provides a simple HTML-based user interface for interaction.

It supports real-time email sending as well as scheduled email functionality using time-based execution. The project demonstrates core concepts of backend development, basic frontend integration, and automation using Python libraries.

This project is designed to simplify repetitive email tasks and improve productivity through automation.
